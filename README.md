# Matrix XP Screensaver

Супер-матрица для Windows XP. Зелёные иероглифы, плавные трассы, режимы "отвал башки".

## Особенности

- Падающие полосы катаканы + цифры + латиница
- Глубинные хвосты с градиентом яркости
- Случайная ерунда и мерцание символов
- Режим **SYSTEM OVERLOAD** — нажмите `E` (безумие + надписи)
- Режим **GLITCH** — нажмите `G` (глитч-артефакты)
- Выход по `ESC`
- Поддержка любых разрешений (автоматически подстраивается)
- Двойная буферизация — без тормозов

## Сборка (MinGW)

```bash
x86_64-w64-mingw32-g++ -mwindows -O2 -Wall src/main.cpp -o matrix.scr -lgdi32 -luser32
```

## Установка

1. Скопируй `matrix.scr` в `C:\Windows\System32\`
2. В настройках экрана (Панель управления → Экран → Заставка) выбери "Matrix XP"
3. Наслаждайся

## Горячие клавиши

- `ESC` — выход
- `E` — включить/выключить "SYSTEM OVERLOAD"
- `G` — включить/выключить глитч-режим

## Техническое

- Монополизация экрана (topmost, WS_POPUP)
- Оптимизация: память, минимальные аллокации
- Кросс-разрешение: column = screenWidth / CHAR_W, row = screenH / CHAR_H
- Без внешних зависимостей (только Win32 API)
